# Solar CCTV Tower — Technical Specification

**Product:** Fyrfly Solar CCTV Tower
**Source:** Fyrfly Systems Ltd — [www.fyrflysystems.com/cctv-tower.html](https://www.fyrflysystems.com/cctv-tower.html)

---

## Mechanical Specification

| Parameter | Value |
|---|---|
| Mast heights | 5m, 6m |
| Mast operation | Pneumatic or manual raise |
| Deployment time | Under 90 minutes (single engineer) |
| Operating temperature | -20°C to +55°C |
| IP rating (mast head) | IP66 |
| IP rating (electronics enclosure) | IP54 |
| Trailer option | Road-legal; pump-truck slots; forklift points |

## Power System

| Parameter | Value |
|---|---|
| Solar array options | 800W, 1000W, 1200W monocrystalline |
| Panel tilt | 35° (optimal for UK latitudes) |
| Panel aspect | South-facing |
| System losses | 14% (wiring, inverter, temperature) |
| Battery chemistry | LiFePO4 (lithium iron phosphate) |
| Battery capacity | 440Ah at 24V = 10.56 kWh total |
| Usable capacity | 8.45 kWh (80% depth of discharge) |
| Daily system draw | ~1.0 kWh |
| Battery autonomy (no solar) | ~8.5 days |
| CO₂ saving vs diesel equivalent | ~550kg per tower per year |

## Connectivity

| Option | Description |
|---|---|
| 4G/5G Cellular | Dual-SIM primary link |
| Wireless PTP | Point-to-point microwave to existing infrastructure |
| Starlink | Satellite broadband for remote or international sites |

## Camera Payload Options (Configurable)

| Camera Type | Application |
|---|---|
| PTZ (pan-tilt-zoom) | Wide-area coverage; target tracking |
| Static bullet | Fixed-zone monitoring |
| 360° panoramic | Open-area coverage; crowd monitoring |
| Thermal imaging | Out-of-hours detection; fog and foliage penetration |

## Security IoT Options

- Radar detection (active perimeter, up to 100m range)
- PIR sensor (passive infrared motion detection)
- Acoustic sensor (gunshot, glass break, anomalous sound)

## Additional Capabilities

- Audio warning (integrated PA horn; AI-triggered or operator-initiated)
- ARC monitoring integration (NSI-approved; visual CCTV verification)
- CO₂e savings monitoring (real-time dashboard; ESG reporting)

## VMS Integration

Fully integrated with Fyrfly VMS. Same management interface as permanent estate. All AI analytics functions available. ARC-connected throughout deployment.
