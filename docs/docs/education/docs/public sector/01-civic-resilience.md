## The Legacy Infrastructure Problem

The typical UK town centre CCTV estate was built on analogue cameras on coaxial cables, recorded to tape or early DVRs, monitored by staff conducting manual screen-watches. These systems:
- Do not communicate with civic building access control systems
- Share bandwidth with public Wi-Fi not designed for security traffic
- Cannot be managed centrally across multiple council sites
- Cannot trigger an automatic lockdown response

## The Fragmentation Failure Mode

When CCTV, access control, and network infrastructure are disconnected:
- A lockdown command cannot propagate because the access control system is not connected to the CCTV management platform
- Cameras go offline due to bandwidth conflicts with public Wi-Fi; discovery is delayed by weeks
- Subject Access Requests take weeks to fulfil because footage is across four separate systems
- Multiple supplier maintenance contracts create accountability gaps when faults cross system boundaries

## The 2026 Convergent Pressures

**Martyn's Law (April 2027 enforcement):** Local authorities with venues or public events of 100+ capacity must have documented protective security procedures. Most town centres, civic halls, libraries, and council-managed public spaces are in scope. Failure to comply is a criminal liability for the responsible person.

**Data (Use and Access) Act 2025:** Enhanced personal accountability for SIROs and DPOs. A CCTV system without a documented DPIA, unclear retention periods, and uncontrolled access rights is a live regulatory exposure, not a legacy problem.

**Budget pressure:** Capital investment must be justified on long-term total cost of ownership, not procurement cost. A single integrated system replacing multiple legacy contracts delivers measurable financial governance benefit.
