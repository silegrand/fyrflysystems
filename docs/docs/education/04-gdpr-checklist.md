## CCTV Deployment — UK GDPR & ICO Requirements

- [ ] Lawful basis documented (typically Public Task or Legitimate Interests; LIA completed where applicable)
- [ ] DPIA completed before deployment or significant expansion; reviewed annually
- [ ] Retention period defined and documented (standard: 31 days; extension process documented for evidential material)
- [ ] Visible ICO-compliant signage at all camera locations
- [ ] Role-based access controls: named individuals only; all access logged and auditable
- [ ] Subject Access Request (SAR) process documented including third-party redaction
- [ ] Data processor agreements in place with all system suppliers
- [ ] Footage encrypted at rest (AES-256) and in transit (TLS 1.3)
- [ ] Incident log maintained: all footage access recorded with purpose, accessor, timestamp
- [ ] Privacy notice in school's published policy covers CCTV processing

## Visitor Management

- [ ] Lawful basis for visitor identity data documented
- [ ] Data minimisation applied: only data necessary for safeguarding collected
- [ ] Retention period for visitor records defined and automated (default: 12 months)
- [ ] Privacy notice displayed at reception
- [ ] Right-to-erasure workflow in place for deletion requests

## Access Control — Credential Data & Audit Trails

- [ ] Credential data (card, fob, mobile) handled under documented lawful basis with retention limits
- [ ] Biometric data: Article 9 UK GDPR special category; explicit consent or documented alternative basis; separate DPIA
- [ ] Access audit trail retained per defined period; searchable for safeguarding investigation
- [ ] Credential revocation process: immediate revocation on staff departure

## Network & Data Security

- [ ] Security network logically or physically separated from student/staff learning network
- [ ] All security data in transit encrypted (TLS 1.3 minimum)
- [ ] Network monitoring in place: alerting on device outages
- [ ] Annual penetration test by CREST-accredited provider
- [ ] Disaster recovery plan covers security system outage scenarios

## ICO Note

The ICO has published specific guidance on CCTV in schools. Key points:
- CCTV must be proportionate to the specific risk it addresses
- Must not be used for purposes beyond those specified in the DPIA
- Staff surveillance requires additional legal justification beyond general safeguarding
- Policies must be reviewed annually
