## Edge-AI Control Room Support

AI analytics flag anomalies to control room operators; humans make decisions. This distinction is material for compliance with the Surveillance Camera Code of Practice, which requires proportionate processing subject to human oversight.

### Analytics Capabilities

| Analytic | Description | Regulatory Note |
|---|---|---|
| Loitering detection | Configurable dwell-time threshold; operator alert with camera snapshot | Proportionate: requires documented pressing need (PoFA 2012) |
| ANPR | Plate capture; PNC and permit database matching | Data retained only as specified in DPIA |
| Illegal dumping | Vehicle stops, deposits material, departs; registration captured for enforcement | Purpose: environmental enforcement, documented in DPIA |
| Crowd formation | Density threshold monitoring; alert on exceedance | Purpose: public safety, counter-terrorism preparedness |
| Abandoned objects | Stationary unattended item against scene baseline | Purpose: counter-terrorism; Martyn's Law supporting measure |
| Thermal intrusion | Out-of-hours person detection, unaffected by lighting | Purpose: asset protection, documented in DPIA |

## Rapid-Deployment Solar CCTV Towers for Events

For temporary public events — Christmas markets, civic festivals, community gatherings — Fyrfly solar CCTV towers extend council monitoring capability to any location without civil infrastructure.

- Deployed in under 90 minutes
- 800W–1200W solar; 440Ah LiFePO4 battery; ~8.5 days autonomy without solar
- 4G/5G connectivity; Starlink optional
- Integrated with council's permanent VMS and ARC
- Martyn's Law compliant for Standard Tier events (100–799 capacity) and supporting measure for Enhanced Tier (800+)

## CONTEST Alignment

The UK counter-terrorism strategy (CONTEST) identifies Protect and Prepare as two of its four principal strands. AI-assisted public space CCTV directly supports both: deterring and identifying threats (Protect); providing control room situational awareness during an unfolding incident (Prepare).
