## What the Audit Covers

### Public Space CCTV Review
- Existing camera coverage, legacy equipment assessment, network dependency risks
- AI analytics upgrade opportunities
- ANPR system compliance status

### Access Control Assessment
- Civic building entry points against current best practice
- Lockdown capability evaluation against Martyn's Law tier requirements

### Network Infrastructure Audit
- Security VLAN separation from public-facing connectivity
- Bandwidth analysis, resilience testing, failover capability review

### Martyn's Law Gap Analysis
- Written assessment of each venue and event type against Standard and Enhanced Tier
- Prioritised compliance roadmap with April 2027 deadline context

### SIRO/DPO Governance Review
- DPIA status across all surveillance systems
- Retention policy review
- SAR process capability against current ICO guidance
- Surveillance Camera Code of Practice self-assessment status

### Financial Case Document
- Five-year total cost of ownership comparison
- Committee-ready investment case for Cabinet or scrutiny committee submission

## Terms

- Standard audit: one site visit per venue type; written report within ten working days
- Available to single local authorities and combined authorities or regional partnerships
- The written report belongs to the authority; no commitment to proceed with Fyrfly is required
